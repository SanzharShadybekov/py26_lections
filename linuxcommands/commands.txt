

Linux commands

ctrl+alt+T - команда для открытия терминала
Mac OS -> в спотлайт ввести 'терминал'

 ~ -> символ домашней директории 
 / -> символ перехода с одной директории в другую, еще и обозначает корневую директорию

.. -> обозначает предыдущий директорию
. -> обозначает текущую директорию

1. cd(change directory) - команда для перехода из одной директорий в другую

2. ls(list) - команда для вывода содержимого текущей директории

3. Mkdir (make directory) - команда для создания директории

4. touch - создает указанный файл

5. rm - удаляет файлы

6. rmdir - удаляет пустые директории 

7. rm -rf - принудительное удаление

8. Pwd (print working directory) - выводит абсолютный путь до текущей директории

9. mv - команда для переименовывания файла или директории, а также для перемещения файлов или директории
	переименовывание: mv john.txt hello.txt
	перемещение: mv john.txt ~/Desktop

10. Ls -a -> команда для вывода всех файлов и директорий в текущей директории

11. cat <filename> - выводит в терминал все содержимое файла

12. echo -  команда для быстрой записи текстовой информации в какой либо файл
			echo 'текст' > название_файла 

13. Nano <path_to_file> - открыывет файл для редактирования в графическом интерфейсе
*если нет такого файла то nano создаст новый файл
